# RLM Research MCP

Deep research over documents, code, and web using RLM (Recursive Language Model) decomposition.

## Install

```bash
pip install rlm-research-mcp
```

## Quick Start

```bash
export RLM_API_KEY=sk-...
rlm-research run "summarize this document" --source doc.pdf
```

## MCP

```bash
claude mcp add rlm-research -- uvx rlm-research-mcp
```
